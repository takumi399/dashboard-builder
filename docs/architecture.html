Dashboard Builder Architecture

System overview — layers, data flow, and technology breakdown

Frontend
  • React 18 + TypeScript
  • Vite (dev server + build)
  • Ant Design 5 (UI components)
  • ECharts 5 (chart rendering)
  • react-rnd (drag & resize)
  • Zustand (state management)
  • React Router DOM v7
  • Axios (HTTP client)
Backend
  • Python 3.11
  • FastAPI (REST framework)
  • SQLAlchemy 2.0 (async ORM)
  • JWT + bcrypt (passlib)
  • Pydantic v2 (validation)
  • python-jose (token handling)
  • CSV parsing + JSON storage
Infrastructure
  • Docker Compose (full stack)
  • SQLite (development)
  • PostgreSQL-ready (prod)
  • GitHub Actions CI
  • TypeScript strict mode
  • Oxlint (code quality)
