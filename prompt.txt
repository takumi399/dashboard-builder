## 目标：添加 Docker 一键生产部署，公司内网即可使用

当前 docker-compose.yml 是开发模式（SQLite + dev server）。现在要加一个生产模式。

### 任务 1：后端支持 PostgreSQL

修改 backend/app/core/config.py：
- DATABASE_URL 默认用 SQLite，但可以通过环境变量覆盖为 PostgreSQL
- 格式：`postgresql+asyncpg://user:pass@host:5432/dbname`
- 添加 POSTGRES_USER, POSTGRES_PASSWORD, POSTGRES_HOST, POSTGRES_DB 等环境变量支持

修改 backend/requirements.txt，添加：
```
asyncpg==0.29.0
```

### 任务 2：后端生产 Dockerfile

创建 backend/Dockerfile.prod：
```dockerfile
FROM python:3.11-slim
WORKDIR /app
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt
COPY . .
EXPOSE 8000
CMD ["uvicorn", "app.main:app", "--host", "0.0.0.0", "--port", "8000", "--workers", "4"]
```

### 任务 3：前端生产构建 + Nginx

创建 frontend/nginx.conf：
```
server {
    listen 80;
    server_name _;
    
    root /usr/share/nginx/html;
    index index.html;
    
    # SPA fallback
    location / {
        try_files $uri $uri/ /index.html;
    }
    
    # API 反向代理到后端
    location /api/ {
        proxy_pass http://backend:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
    
    # WebSocket 代理
    location /ws/ {
        proxy_pass http://backend:8000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
    }
}
```

创建 frontend/Dockerfile.prod（多阶段构建）：
```dockerfile
# 阶段1：构建
FROM node:20-alpine AS builder
WORKDIR /app
COPY package*.json ./
RUN npm ci
COPY . .
RUN npm run build

# 阶段2：Nginx 托管
FROM nginx:alpine
COPY --from=builder /app/dist /usr/share/nginx/html
COPY nginx.conf /etc/nginx/conf.d/default.conf
EXPOSE 80
CMD ["nginx", "-g", "daemon off;"]
```

### 任务 4：生产 docker-compose

创建 docker-compose.prod.yml：
```yaml
services:
  db:
    image: postgres:16-alpine
    environment:
      POSTGRES_USER: dashboard
      POSTGRES_PASSWORD: dashboard123
      POSTGRES_DB: dashboard
    volumes:
      - pgdata:/var/lib/postgresql/data
    restart: unless-stopped

  backend:
    build:
      context: ./backend
      dockerfile: Dockerfile.prod
    environment:
      DATABASE_URL: postgresql+asyncpg://dashboard:dashboard123@db:5432/dashboard
      SECRET_KEY: change-me-in-production
    depends_on:
      - db
    restart: unless-stopped

  frontend:
    build:
      context: ./frontend
      dockerfile: Dockerfile.prod
    ports:
      - "80:80"
    depends_on:
      - backend
    restart: unless-stopped

volumes:
  pgdata:
```

## 生产部署
