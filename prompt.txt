### 任务 1：后端 XSS 防护

修改 backend/app/schemas/ 中所有涉及用户输入字符串的 Schema：
- DashboardCreate: name 加 max_length=100, 拒绝包含 < > 的输入
- ChartCreate: title 加 max_length=100, 同样拒绝 < >
- Auth 注册: username/email 也拒绝 HTML 标签

在 backend/app/core/ 新建 security.py 添加工具函数：
```python
import re

def sanitize_text(value: str) -> str:
    """移除 HTML 标签，防止 XSS。"""
    return re.sub(r'<[^>]*>', '', value)

def contains_html(value: str) -> bool:
    """检查是否包含 HTML 标签。"""
    return bool(re.search(r'<[^>]*>', value))
```

在 validators 或 API 层调用这些函数。

### 任务 3：密码强度校验

修改 backend/app/schemas/auth.py：
- 注册时的 password 加自定义 validator：
  - 至少 8 个字符
  - 必须包含字母和数字
  - 返回 422 并给出明确的错误提示

```python
@field_validator('password')
@classmethod
def validate_password_strength(cls, v):
    if len(v) < 8:
        raise ValueError('密码至少需要 8 个字符')
    if not re.search(r'[a-zA-Z]', v):
        raise ValueError('密码必须包含字母')
    if not re.search(r'[0-9]', v):
        raise ValueError('密码必须包含数字')
    return v
```
