### 任务 3：数据库自动备份

创建 backend/app/core/backup.py：
- backup_database() 函数：用 pg_dump 导出 SQL 到 backup/ 目录
- 文件名带时间戳：backup_20260101_120000.sql
- 返回备份文件路径

创建备份 API（在 monitor.py 加一个端点）：
- POST /api/admin/backup：手动触发备份（需认证）
- GET /api/admin/backups：列出历史备份文件

创建 scripts/backup.sh：
```bash
#!/bin/bash
# 定时备份脚本，配合 cron 使用
TIMESTAMP=$(date +%Y%m%d_%H%M%S)
BACKUP_DIR="./backups"
mkdir -p $BACKUP_DIR
pg_dump $DATABASE_URL > "$BACKUP_DIR/backup_$TIMESTAMP.sql"
# 保留最近 7 天的备份
find $BACKUP_DIR -name "*.sql" -mtime +7 -delete
```

同时在 requirements.txt 添加 structlog 和 slowapi。
